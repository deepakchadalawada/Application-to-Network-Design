Readme filefor the project ""AN APPLICATION TO NETWORK DESIGN""
-----------------------------------------------------------------
The source file is written in JAVA and the following steps needs to be followed to execute the code:
1. Create a JAVA Project with the Class name ATNProject1
2. The source file is then to be compiled and executed without giving any input.
3. The number of Nodes 'N' for the Network is hard coded as 30
4. The value of K is also hard coded and this ranges between 3 and 15.
5. The program gives the Cost and Density for all the K values from K=3 to K=15. 
6. These are created in a tabular format which can be uploaded to Excel sheet to draw graphs.
